# Arrotech - AI Solutions Company Website

A modern, responsive website for Arrotech, showcasing our AI solutions and services. Built with React, TypeScript, and Tailwind CSS.

## Features

- Modern and responsive design
- Smooth animations and transitions
- Interactive components
- Mobile-first approach
- SEO-friendly structure
- Fast loading times

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Vite

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/arrotech.git
cd arrotech
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and visit `http://localhost:5173`

### Building for Production

To create a production build:

```bash
npm run build
# or
yarn build
```

The build files will be created in the `dist` directory.
